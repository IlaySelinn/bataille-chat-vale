# 🐈 Bataille Chat-vale

Un jeu de bataille navale revisité avec des chats, développé en **JavaFX**.

## 🚀 Comment jouer ?
1. Téléchargez le fichier `.jar` dans l'onglet **Releases**.
2. Assurez-vous d'avoir **Java 21** installé.
3. Double-cliquez et coulez (ou endormez) les chats de l'IA !

## 🛠️ Côté Technique
Ce projet m'a permis de travailler sur :
- **JavaFX** : Interface graphique et gestion d'événements.
- **POO** : Utilisation de l'héritage pour créer différents niveaux d'IA.
- **Gestion du son** : Intégration de sons `.wav` en temps réel.

## Fonctionnalités
* Placement de chats (Tom, Pufi, etc.) sur une grille 10x10.
* Effets sonores (miaulements et ronronnements).
* Deux niveaux d'IA :
    * **Facile** : Tire au hasard.
    * **Difficile** : Chasse intelligemment dès qu'un chat est touché.

## Comment lancer le jeu ?
1. Cloner le projet.
2. Importer le projet Maven dans votre IDE (IntelliJ/Eclipse).
3. Lancer la classe `MainMenu`.
